Wi-Fi room commands in Kid A:

NOTE: These commands can only be used in the Wi-Fi room or in PM.

Cloner list:

addcloner (username, ign, fc, notes): Adds a cloner to the cloner list. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Requires Room Owner or cloner whitelist.

removecloner (username): Removes a cloner from the cloner list. Requires Room Moderator.

updatecloner (username, [key: value, key: value]): Updates the information of a cloner. Possible values for keys are: fc, ign, notes. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Cloners can update their own info, and Room Owners or cloner whitelisted users can edit any cloner's info.

clonerga (username): Approves a cloner's monthly giveaway, and updates the date on the list. Requires Room Owner or cloner whitelist.

purgecloners: Removes all cloners from the list that haven't done a giveaway in the past 30 days. Requires Room Owner.

whitelistcloner (username): Adds the given user to the cloner whitelist, allowing them to use cloner commands. Requires Room Owner.

Trainer list:

addtrainer (username, ign, fc, evs, levels, collateral, notes): Adds a trainer to the trainer list. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Requires Room Owner or trainer whitelist.

removetrainer (username): Removes a trainer from the trainer list. Requires Room Moderator.

updatetrainer (username, [key: value, key: value]): Updates the information of a trainer. Possible values for keys are: fc, ign, evs, levels, collateral, notes. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Trainers can update their own info, and Room Owners or trainer whitelisted users can edit any trainer's info.

traineractivity (username): Approves a trainer's monthly EV training, and updates the date on the list. Requires Room Owner or trainer whitelist.

purgetrainers: Removes all trainers from the list that haven't done a giveaway in the past 30 days. Requires Room Owner.

whitelisttrainer (username): Adds the given user to the trainer whitelist, allowing them to use trainer commands. Requires Room Owner.

Scammer list:

addscammer (main, alts, ign, fc, evidence, reason): Adds a scammer to the scammer list. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Requires Room Driver.

removescammer (username): Removes a scammer from the scammer list. Requires Room Moderator.

updatescammer (username, [key: value, key: value]): Updates the information of a scammer. Possible values for keys are: alts, fc, ign, evidence, reason, addedby. If you wish to have commas in any of the fields, use '|' as the separator instead of ','. Room Drivers can edit any scammer's info.

addscammeralt (username, alt1[, alt2, alt3...]): Adds an alt to the scammer's entry. Requires Room Driver.

checkfc (fc): Checks if the given FC is on the scammer list.
